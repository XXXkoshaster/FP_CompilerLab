# FScheme — интерпретатор Scheme на F#

## Запуск

```bash
dotnet run -- examples/factorial.scm
```

## Синтаксис языка

Scheme использует S-выражения — всё записывается в скобках:

```scheme
; Комментарии начинаются с ;

; Арифметика (префиксная запись)
(+ 1 2)          ; => 3
(* 2 (+ 3 4))    ; => 14

; Переменные
(define x 42)

; Функции (лямбды)
(define square (lambda (x) (* x x)))
(square 5)        ; => 25

; Условие
(if (> x 0) x (- 0 x))

; Локальные привязки
(let ((a 1) (b 2)) (+ a b))

; Рекурсия через letrec
(letrec ((fact (lambda (n)
  (if (= n 0) 1 (* n (fact (- n 1)))))))
  (fact 5))

; Последовательность выражений
(begin (display 1) (newline) (display 2) (newline))

; Списки
(list 1 2 3)
(cons 1 (list 2 3))  ; => (1 2 3)
(car (list 1 2 3))   ; => 1
(cdr (list 1 2 3))   ; => (2 3)
(null? (list))       ; => 1
```

## Встроенные функции

| Функция | Описание | Пример |
|---------|----------|--------|
| `+` `-` `*` `/` `%` | Арифметика | `(+ 2 3)` → `5` |
| `=` `>` `<` `>=` `<=` | Сравнение (1/0) | `(> 3 2)` → `1` |
| `if` | Условие | `(if cond then else)` |
| `lambda` | Создать функцию | `(lambda (x) (* x x))` |
| `define` | Глобальное определение | `(define x 5)` |
| `let` | Локальные привязки | `(let ((x 1)) x)` |
| `letrec` | Рекурсивные привязки | `(letrec ((f ...)) (f 5))` |
| `begin` | Последовательность | `(begin e1 e2 e3)` |
| `cons` | Добавить в голову | `(cons 1 (list 2))` |
| `car` | Голова списка | `(car (list 1 2))` → `1` |
| `cdr` | Хвост списка | `(cdr (list 1 2))` → `(2)` |
| `list` | Создать список | `(list 1 2 3)` |
| `null?` | Пустой список? | `(null? (list))` → `1` |
| `display` | Вывести на экран | `(display 42)` |
| `newline` | Перевод строки | `(newline)` |
| `read-file` | Прочитать файл | `(read-file "a.txt")` → строка |
| `write-file` | Записать в файл | `(write-file "a.txt" "hi")` |
| `append-file` | Дописать в файл | `(append-file "a.txt" "more")` |
| `file-exists?` | Файл существует? | `(file-exists? "a.txt")` → `1` |
